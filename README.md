Create HTML tags and render them efficiently.

### Requirements
    
    - PHP >= 5.4.0
    
### Composer

You can install the package via [Composer](http://getcomposer.org/). Add this to your `composer.json`:

    {
      "require": {
        ...
        "arcanedev/markup": "dev-master"
        ...
      }
    }
    
Then install via:

    composer.phar install
